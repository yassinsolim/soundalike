Ratings stay in this browser. There is no analytics or server submission. When you request audio, the preview API receives only the numeric Deezer track ID—never rating values, rater ID, session ID, or localStorage data. The fresh signed preview URL stays only in this page's memory.

\r\n

A complete session is about 480 unique result ratings plus 120 list judgments over 60 seeds (usually 90–150 minutes). Stop anytime: progress autosaves only in this browser, and partial exports can be resumed. Export the ratings JSON and send it to the project owner yourself; this page never uploads it. Use at least 3 independent raters (ideally 5) for population evidence.
