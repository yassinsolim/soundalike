Blind served-list listening study

Ratings autosave in this browser. Nothing is submitted automatically. When you request audio, the preview API receives only the numeric Deezer track ID—never rating values, rater ID, session ID, or localStorage data. The fresh signed preview URL stays only in this page's memory.

A complete session is about 480 unique result ratings plus 120 list judgments over 60 seeds (usually 90–150 minutes). Stop anytime: partial progress can be resumed, submitted with explicit consent, or downloaded with Export JSON as a manual fallback. Use at least 3 independent raters (ideally 5) for population evidence.

Submit ratings privately

Submission is anonymous and manual. The upload contains random anonymous/session IDs, complete rating values, rating timestamps and interaction durations, locked protocol/list hashes, and the local HMAC key and HMAC used only to verify this snapshot.

The private stored record keeps the rating evidence but strips the local HMAC key and HMAC. It does not store an IP address, Origin, browser/user-agent, cookies, Spotify identity, email, raw headers, or a private Blob URL. Vercel may still process request metadata in operational infrastructure; the application does not copy it into the ratings record.
